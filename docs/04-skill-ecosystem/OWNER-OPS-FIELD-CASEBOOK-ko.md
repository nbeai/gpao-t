# Owner Ops Field Casebook

Status: v0.1 local baseline  
Scope: Korean owner-operator no-API automation workflows

## 목적

`Owner Ops Pack`은 한국 자영업자가 자동화 도구를 설계하지 않아도, 자기 사업의 반복 문제를 말하면 안전한 자동화 후보와 초안을 얻도록 만든다.

핵심 원칙:

```text
사용자는 자동화를 만들지 않는다.
사용자는 사업 문제를 말한다.
GPAO-T가 반복 업무, 권한 경계, 초안, 기록, 개선 후보로 번역한다.
```

## 대표 업종과 반복 업무

| 업종 | 반복 업무 | 안전 시작 |
| --- | --- | --- |
| 음식점 / 카페 | 리뷰 답변, 예약 문의, 불만 응대, 메뉴/영업시간 안내 | 리뷰/문의 붙여넣기 요약과 답변 초안 |
| 미용 / 네일 / 피부관리 | 예약 문의, 가격 안내, 노쇼 방지, 후기 답변 | 예약 문의 초안과 확인 질문 |
| 학원 / 레슨 / 코칭 | 상담 문의, 수업 일정, 결석/보강, 학부모 응대 | 상담 문의 분류와 답변 초안 |
| 쇼핑몰 / 스마트스토어 | 배송 문의, 교환/환불, 재입고, 상품 정보 | 문의 CSV/붙여넣기 분류와 초안 |
| 병원 / 상담 / 예약 서비스 | 예약 문의, 준비물, 주의사항, 변경/취소 | 예약 가능성 확인 전 초안 |
| 공방 / 스튜디오 / 프리랜서 | 견적 문의, 예약 가능일, 작업 범위, 시안/수정 안내 | 견적/예약 문의 분류와 추가 질문 |

## 첫 사용자 문장

- 리뷰 답변이 밀려요.
- 예약 문의가 계속 와서 놓쳐요.
- 스마트스토어 문의를 분류하고 답장 초안을 만들고 싶어요.
- 부정 리뷰만 따로 보고 싶어요.
- 이번 주에 반복된 고객 불만을 보고 싶어요.

## 필드 리스크

- 고객에게 바로 보내는 자동 발송은 v0.1에서 차단한다.
- 환불, 결제, 삭제, 대량 메시지는 자동화하지 않는다.
- 전화번호, 계좌, 주민번호처럼 민감해 보이는 값은 초안/기록에서 주의 표시한다.
- 법률/세무 확정 판단은 하지 않는다.
