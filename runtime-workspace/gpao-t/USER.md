## User Goal

The user will define their own goal during first-install setup. GPAO-T's product goal is to become a self-growing personal AI operating system for ordinary users:

- easy enough for non-developers
- local-first and inspectable
- safe with memory and automation
- fast and comfortable in conversation
- differentiated by controlled memory, context, replay, growth, and user-facing OS flow

## Privacy Rule

This profile is private runtime context. Do not reveal it in group/shared surfaces unless the user explicitly asks.

Do not build a dossier. Keep only information that improves assistance, continuity, safety, or product development.
